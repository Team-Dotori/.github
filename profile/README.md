### Golabab 이란?
GSM 교내 급식 선호도 조사 서비스입니다.
- 기존 GooglaForm 을 통한 선호도 조사서비스의 불편을 해결하고
- 학생들도 투표 결과를 직접 볼 수 있도록 

개발하였습니다
